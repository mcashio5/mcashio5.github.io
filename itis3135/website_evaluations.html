Website Evaluations

Applying the CRAP principles: Contrast, Repetition, Alignment, Proximity.

Awwwards (Good Example)

Awwwards showcases high-quality web design with a clean, modern interface and strong editorial curation.

  • Contrast: Clear type hierarchy, generous whitespace, and deliberate color accents make important content pop.
  • Repetition: Consistent typography, spacing, and card/grid patterns create cohesion across pages.
  • Alignment: A precise grid keeps imagery, copy, and controls neatly aligned, improving scanability.
  • Proximity: Related elements (title, meta, actions) are grouped, so users instantly see what belongs together.

Arngren.net (Crappy Example)

Arngren is an infamous catalog page with dense, unstructured blocks of images and text competing for attention.

  • Contrast: Clashing colors and crowded visuals create noise instead of focus; nothing stands out meaningfully.
  • Repetition: Inconsistent fonts, sizes, and ad-hoc modules break visual rhythm and make the UI feel chaotic.
  • Alignment: Elements appear scattered with minimal grid discipline, producing a jagged reading path.
  • Proximity: Unrelated items are jammed together; related content isn’t grouped, which hurts comprehension.
